**Form (`<form id="workshopForm">`):**

Hidden inputs (direct children of form, ABOVE the workshopFields div):
```html
<input type="hidden" name="access_key" value="YOUR_WEB3FORMS_ACCESS_KEY">
<input type="hidden" name="subject" value="Workshop Inquiry -- Redcocoon">
<input type="hidden" name="from_name" value="Redcocoon Website">

```

`<div id="workshopFields">` containing:
1. Name field: `<input type="text" name="name" required placeholder="Your name" />`
2. Email field: `<input type="email" name="email" required placeholder="your@email.com" />`
3. Phone field: `<input type="tel" name="phone" placeholder="Phone or WhatsApp (optional)" />`
4. Message field: `<textarea name="message" required rows="4" placeholder="Tell us about what you're interested in -- a weekend session, a private class, a group event..." />`
5. hCaptcha div: `<div class="h-captcha" data-captcha="true"></div>`
6. Submit button: `<button type="submit" class="btn btn-primary workshop-submit">Send Inquiry</button>`

All inputs must have min-height 44px (touch target). Use consistent field styling: background rgba(255,255,255,0.6), border 1px solid var(--color-beige-dark), border-radius var(--border-radius), padding 0.8rem 1rem, font-size var(--text-base), color var(--color-espresso). Focus state: border-color var(--color-terracotta), outline none.

Success panel (`<div id="workshopSuccess" style="display:none; opacity:0;">`):
- Green checkmark icon (use CSS circle with checkmark, colored var(--color-sage))
- Text: "Thanks! We'll be in touch soon."
- Styled as flex column centered, gap var(--space-md)

Error line (`<p id="workshopError" class="workshop-error" style="display:none;"></p>`):
- Red/terracotta text, centered, margin-top var(--space-md)

**Vanilla JS `<script>` (standard, NOT is:inline):**

Follow the exact Web3Forms fetch pattern from RESEARCH.md Pattern 2:
1. Get form, fields div, success div, error element by ID
2. `form.addEventListener('submit', async (e) => { ... })`
3. `e.preventDefault()`
4. Create FormData, convert to object with Object.fromEntries
5. POST to `https://api.web3forms.com/submit` with JSON content-type
6. On 200: fade out workshopFields (opacity 0), then after 300ms set display:none, show workshopSuccess with opacity transition
7. On non-200: show error message from json.message or fallback "Something went wrong. Please try again."
8. On catch: show "Network error -- please check your connection and try again."

**External CDN script (use `is:inline` attribute to prevent Astro bundling):**
```html
<script is:inline src="https://web3forms.com/client/script.js" async defer></script>
```
Place this AFTER the form, before the closing of the page content. The `is:inline` is required for external CDN scripts in Astro to prevent bundler interference.
